    03 · Evolution

    EUC Evolution Timeline

    Click any era to expand its detail. Scroll horizontally to traverse six decades of computing architecture.

    The Mainframe Age · 1960s–1970s

    Architecture

    Room-sized machines running batch jobs. Dumb terminals connected via coaxial. Zero local processing. IBM System/360 defined the model: one OS across hardware sizes.

    Operational Challenge

    Cost prohibitive. Multi-million dollar installations. Only governments, banks, and universities could afford access. IT was a priesthood controlling all compute resources.

    Why It Matters Today

    Philosophically identical to cloud desktops: powerful compute in a datacentre, a thin device at the edge. The architecture was right; the network didn't exist yet.

    Key Vendors

    IBM (dominant), DEC, Honeywell, Burroughs, UNIVAC, NCR. Known collectively as "IBM and the BUNCH."

    The PC Revolution · 1980s

    Architecture

    IBM PC (1981) with open architecture. Microsoft licensed MS-DOS to every clone manufacturer. Compute decentralised to every desk. Local storage, local processing, local chaos.

    Operational Challenge

    Shadow IT born. No backups, no software inventory, no patch management. Viruses spread via floppy disk. IT went from managing one machine to managing thousands with no tools.

    The DNA Moment

    Created the central paradox of all EUC: the more freedom users have, the harder IT's job becomes. Every subsequent era attempts to rebalance this equation.

    Key Vendors

    Microsoft (dominant), Apple, Novell (NetWare), Intel, Lotus (1-2-3), WordPerfect. Atari, Commodore, Tandy all died.

    Client-Server & Citrix · 1990s

    Architecture

    LANs connected PCs. Client-server emerged. Citrix (founded 1989) invented multi-user Windows delivery. WinFrame shipped 1995. Microsoft licensed the tech for Terminal Services.

    Operational Challenge

    Physical visits for every patch. Primitive tools (SMS, LANDesk). Internet arrived (1993-95) creating new security and delivery questions. Browser became a thin client.

    The Citrix Insight

    Why manage software on 1,000 PCs when you can run it once on a server? This insight proved so durable it underpins modern cloud desktops 30 years later.

    Key Vendors

    Citrix, Microsoft, Novell NetWare, Cisco, Sun Microsystems, 3Com. Netscape and WordPerfect declined.

    The VDI Era · 2000–2012

    Architecture

    VMware made server virtualisation mainstream. VDI: virtual desktops in on-prem datacentres. One golden image, infinite desktops. Citrix XenDesktop vs VMware View.

    Operational Challenge

    $1M+ deployments, 18-month timelines, 6-8 specialist skills required, ~40% failure rate. Boot storms, storage I/O, profile corruption. A cottage industry of fixes emerged.

    The Profile Problem

    FSLogix (founded 2012, acquired by Microsoft 2018) solved VDI's worst pain with profile containers. Now free with every M365 licence and core to every AVD deployment.

    Key Vendors

    Citrix XenDesktop, VMware View/Horizon, Microsoft RDS, Wyse, AppSense, RES Software, Liquidware, Quest.

    Cloud Transition · 2013–2019

    Architecture

    Nadella's cloud-first pivot (2014). SaaS displaced on-prem apps. Identity replaced the network perimeter. Hybrid everything. Azure AD, Intune, Conditional Access built.

    Operational Challenge

    Perimeter dissolved. SD-WAN needed for SaaS performance. Zero Trust philosophy required new tooling. Mobile/BYOD added management surface. DEX monitoring emerged.

    Zero Trust

    John Kindervag (Forrester, 2010): "Never trust, always verify." Identity, device health, context, and risk signals evaluated for every access request. Microsoft's Conditional Access became the practical implementation.

    Key Vendors

    Microsoft (Azure, O365, Intune), Zscaler, Palo Alto, Okta, VMware VeloCloud, Lakeside SysTrack.

    Cloud Desktop · 2020–2024

    Architecture

    COVID compressed 5 years into 3 months. AVD (Sept 2019) gained massive urgency. Windows 365 (Aug 2021) democratised Cloud PCs. Infrastructure as Code replaced GUIs.

    Operational Challenge

    Cloud solved infrastructure. Created operational complexity: cost management, scaling, image lifecycle, multi-tenant orchestration. The complexity shifted, not decreased.

    The Broadcom Disruption

    Broadcom acquired VMware (Oct 2023). Prices up 10x. Perpetual licences eliminated. Omnissa spun off. Largest platform migration wave in EUC history. Microsoft and Nerdio primary beneficiaries.

    Key Vendors

    Microsoft (AVD, W365), Nerdio (NME, NMM), ControlUp, Nexthink, Citrix DaaS. Nerdio reached $1B+ valuation.

    AI Workspace · 2025+

    Architecture

    AI-augmented operations. Physical and virtual device convergence. Copilot embedded across M365. The EUC platform becomes the secure delivery vehicle for AI-enabled work.

    Operational Challenge

    AI governance, data residency, autonomous operations trust. Managing AI tools safely at scale on compliant infrastructure. New skill requirements for EUC teams.

    Autonomous Operations

    Auto-scaling was step one. Auto-heal was step two. AI-driven predictive operations is step three. EUC administrators will manage policies and exceptions, not individual VMs.

    Key Vendors

    Microsoft, Nerdio (Scott Manchester as CPTO from Jan 2026), AI-native tooling, converged management platforms.

    06 · Future

    The Future of EUC

    Five trajectories shaping the next phase. Click to expand.

    01
    AI-Enabled Workspace Operations
    Copilot embeds AI across M365. The EUC platform becomes the secure delivery vehicle for the AI-enabled workforce. Operations platforms will use AI for predictive scaling, anomaly detection, and automated remediation. Managing AI tools safely — data residency, governance, compliance — becomes a core EUC responsibility.
    02
    Physical & Virtual Convergence
    The boundary between a physical PC managed by Intune and a Cloud PC managed by Windows 365 is dissolving. Unified endpoint management treats all devices — physical, virtual, Cloud PC — as first-class objects in a single policy framework. The user should not know or care whether their desktop is local or cloud-hosted.
    03
    Platform Consolidation
    The Broadcom disruption accelerated consolidation already underway. Microsoft's cloud desktop stack (AVD + Windows 365 + Intune + Entra ID) is the gravitational centre. Citrix retains enterprise presence but faces headwinds. Omnissa's future is uncertain. The ecosystem consolidates around Microsoft, with Nerdio providing the operational layer.
    04
    DEX as a Business Metric
    Digital Employee Experience will evolve from IT monitoring to a business performance metric. Boards will measure technology effectiveness through experience scores, not uptime. DEX data will drive investment decisions: which offices need better connectivity, which applications cause friction, which user segments are underserved.
    05
    Autonomous Operations
    The logical endpoint of automation: environments that monitor themselves, detect anomalies, remediate issues, scale resources, and optimise costs without human intervention. Auto-scaling was step one. Auto-heal was step two. AI-driven predictive operations is step three. The EUC administrator of the future manages policies and exceptions, not individual VMs.
