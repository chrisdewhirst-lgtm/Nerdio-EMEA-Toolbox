# EMEA Nerdio Toolbox

A curated collection of **22 ready-to-use sales tools**, **17 build-your-own templates**, **8 EUC ecosystem references**, and **2 AI skills** for the EMEA Nerdio team. Built to accelerate every stage of the sales cycle — from territory planning to close.

> **Internal Use Only** — This repo contains competitive intelligence, sales positioning, and internal tooling. Do not share externally.

---

## Quick Start

1. **Clone the repo:**
   ```bash
   git clone https://github.com/chrisdewhirst-lgtm/Nerdio-EMEA-Toolbox.git
   ```
2. **Open the hub:** Open `EMEA-Nerdio-Toolbox.html` in your browser — it's the interactive portal with search, filtering, and links to every tool.
3. **Use tools directly:** Green "Ready to Use" tools are HTML files — just open them in any browser. No install, no server needed.
4. **Build your own:** Orange "Build Template" tools are MD instruction files. Upload them to [Claude](https://claude.ai) with your territory data and it builds a personalised version.

## How to Use the Build Templates

1. Open the MD file for the tool you want
2. Follow the "What You Need" section to gather your data
3. Open Claude (claude.ai, Claude Desktop, or Cowork)
4. Upload the MD file and your data files
5. Copy the prompt from the "Prompt for Claude" section
6. Claude will generate a personalised HTML tool you can save and use
