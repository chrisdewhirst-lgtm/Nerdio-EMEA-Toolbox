📚 How to Use the Pseudonym Manager

🔒 Why Pseudonymise?

When using AI tools like Claude, ChatGPT, or any external AI, you should never paste real customer names, deal values, contact details, or sensitive business data. The Pseudonym Manager lets you swap all of that for safe aliases before the data leaves your machine — and swap it back afterwards.

⚡ Mode 1: Pseudonymise Text

Use this when you're about to paste text INTO an AI tool.

1
Paste your raw text into the left panel. The tool auto-detects company names, people, phone numbers, emails, and monetary values.
2
Review the detections. Check the mapping table — dismiss any false positives, add anything it missed.
3
Click the arrow button to pseudonymise. The right panel shows the safe version.
4
Copy and paste the safe output into your AI tool.

📄 Mode 2: Pseudonymise File

Use this to pseudonymise files — especially Excel territory lists, CSV exports, or Word docs with customer data. The tool is column-aware for structured files.

1
Upload your file. For Excel/CSV, the tool reads all columns and auto-suggests which ones contain sensitive data (e.g. "Account Name", "Account Owner", "Contact").
2
Select columns to pseudonymise. Click the column chips to toggle. Only selected columns will be changed — everything else stays untouched.
3
Preview the result — see exactly what changes before downloading. Each unique value gets its own pseudonym.
4
Download the pseudonymised file (Excel stays as Excel!) and use it with AI freely.
5
Save your workspace — you'll need these mappings later to reverse the AI's output.

🔄 Mode 3: Reverse Text

Paste AI output, click reverse, get real data back.

📄 Mode 4: Reverse File

Upload an AI-generated file to swap pseudonyms back. Supports Excel, CSV, Word, HTML, and text files.

💡 Typical End-to-End Workflow

1
Pseudonymise your source data (text or file).
2
Save your workspace (.nps file).
3
Use AI freely with the pseudonymised data.
4
Load your workspace back into the tool.
5
Reverse the AI output to swap pseudonyms back to real data.

💾 Workspaces

Save mappings as encrypted .nps files. Store securely (OneDrive, local folder). Load anytime to resume — no need to remap.

⚠️
Important: All operations happen locally in your browser. No data is sent anywhere. Never paste real customer data into external AI tools. Store .nps workspace files securely.
