# EUC Atlas – Claude Co‑Work Project

This project turns the provided **EUC Evolution HTML asset** into a complete set of assets:

• EUC Evolution Narrative  
• EUC Universe Map (industry architecture)  
• Vendor Landscape  
• Modern Cloud Desktop Reference Architecture  
• Partner Narrative  
• Interactive Microsite Structure

The goal is to use **Claude Co‑Work** to generate:

- diagrams
- Figma layouts
- partner enablement assets
- microsite structure
- presentation content

---

# Step 1 – Upload Files to Claude Co‑Work

Upload **all files in this folder**, including:

- EUC_EVOLUTION_SOURCE.html
- EUC_EVOLUTION_SUMMARY.md
- EUC_UNIVERSE_MAP.md
- EUC_VENDOR_LANDSCAPE.md
- EUC_ARCHITECTURE_REFERENCE.md
- EUC_PARTNER_NARRATIVE.md
- CLAUDE_MASTER_PROMPT.md

Claude will use them as context for generating diagrams and assets.

---

# Step 2 – Run the Master Prompt

Open `CLAUDE_MASTER_PROMPT.md` and paste it into Claude.

Then iterate with prompts like:

• Generate Figma layout for EUC Universe Map  
• Create partner presentation from these docs  
• Generate interactive microsite structure  
• Create architecture diagrams

---

# Step 3 – Build Assets

Recommended tools:

Figma → diagrams  
Gamma → partner presentation  
Framer/Webflow → interactive site  
Canva → partner collateral
