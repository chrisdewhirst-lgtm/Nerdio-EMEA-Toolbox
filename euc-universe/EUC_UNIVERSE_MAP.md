# EUC Universe Map

The EUC ecosystem can be visualised as a layered architecture.

## Users / Workloads

Actors interacting with computing environments.

Examples:

- employees
- contractors
- developers
- AI agents

---

## Devices

Endpoints used to access the workspace.

Examples:

- Windows PCs
- macOS devices
- iOS / Android
- thin clients
- web browsers

---

## Identity & Security

Controls authentication and access.

Key technologies:

- Entra ID
- Conditional Access
- MFA
- Zero Trust

---

## Desktop Platform

Delivers Windows desktops remotely.

Platforms:

- Azure Virtual Desktop
- Windows 365
- Citrix
- Omnissa Horizon
- AWS Workspaces

---

## Operations & Automation

Tools that automate cloud desktop environments.

Capabilities:

- provisioning
- scaling
- cost optimisation
- automation

Key Platform:

Nerdio

---

## Application Delivery

Technologies used to deliver applications.

Examples:

- MSIX
- App Attach
- App Volumes
- application streaming
- SaaS

---

## Endpoint Management

Manages device configuration.

Platforms:

- Microsoft Intune
- Configuration Manager
- Jamf
- Workspace ONE

---

## Digital Employee Experience (DEX)

Measures user experience.

Platforms:

- ControlUp
- Nexthink
- Lakeside
- Aternity
- Rimo3

---

## Infrastructure

Underlying cloud infrastructure.

Providers:

- Microsoft Azure
- AWS
- Google Cloud
