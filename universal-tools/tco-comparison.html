Deployment types

How the recommendation engine works

AVD Pooled
Multi-session shared VMs. Best for knowledge/task workers with cloud-ready apps, high concurrency, and variable hours. Nerdio auto-scaling delivers 30–50%+ compute saving.
AVD Personal
Dedicated VM per user. Required for legacy/client-server apps, GPU workloads, or users who need persistent desktop state. Still benefits from Nerdio auto-shutdown and Spot/RI blending.
W365 Enterprise
Fixed Cloud PC per user. Best for standard knowledge workers, simple app stacks, predictable usage, and teams that need cost predictability over optimisation.
W365 Frontline
1 licence = 3 non-concurrent users. Best for shift workers, seasonal staff, contractors, or anyone with <6hrs/day average usage. ~⅓ the per-user cost of standard W365.
Local Compute
User must run locally. Triggered by: hardware-dependent apps, USB peripherals, low connectivity, real-time audio/video production, or offline requirements.
Hybrid
Mix of cloud and local. Nerdio manages both cloud desktops and Intune-enrolled physical endpoints from a single pane — unique capability.
